---- Prerequisites for using 7gen:

*   Python (>=2.3.x. Tested with 2.3.4, but later versions should work.)
You can get python at http://www.python.org/

*   PyGame (>=1.6. Tested with 1.6, later versions should work)
You can get pygame at http://www.pygame.org/

Other requirements: 

JKL2VXP:
To use jkl2vxp, you need a copy of Dark Forces 2: Jedi Knight or 
Jedi Knight: Mysteries of the Sith. You can buy Jedi Knight at:
http://www.lucasarts.com/companystore/jediknight/ 
I haven't been able to find a buy link for MOTS. (Check eBay or Yahoo Auctions.)

OBJ2VXP:
OBJ2VXP is designed for use with OBJ files created by PlantStudio 2. Other
OBJ files might work, but haven't been tested. You can get PlantStudio 2 here:
http://www.kurtz-fernhout.com/PlantStudio/
Make sure you check out the Order page (http://www.kurtz-fernhout.com/order.htm),
to get a free serial number for PlantStudio 2. 

ACTOREXTRACTOR:
ActorExtractor requires a copy of 3D Movie Maker to be installed and NoCD'd. 
You can get a utility that NoCDs 3DMM here:
http://v3dmm.com/downloads/3dmmNoCD.exe
(Requires the original 3DMM CD)
Also, the automatic combiner scripts created by ActorExtractor are currently Windows-only.
(They use InfoZip's ZIP.EXE to create the .VXP at high compression)
